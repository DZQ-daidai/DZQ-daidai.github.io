Ubuntu下安装Node.js

  • 配置npm

    • sudo apt update
    • sudo apt install npm

  • 升级npm

    • sudo npm i -g npm

  • 安装Node.js

    • sudo apt install nodejs

  • 升级Node.js

    • 利用n来管理版本

    • n来下载node版本

    • 切换版本

    • 或者直接指定版本

    • 查看node版本
